## 光照风格
- **核心光源**：生物荧光 — 来自记忆碎片和环境中的微弱发光体
- **环境光**：极低，整体偏暗，营造深渊压迫感
- **特殊光效**：
  - 记忆之锚：脉冲式荧光青发光，有呼吸感
  - 侵蚀效果：边缘泛虚空橙，像现实被烧穿
  - 宇宙存在：不规则的光线扭曲，引力透镜效果
- **光照参考**：深海生物荧光 + 星云摄影 + Hollow Knight 的"光点"系统

## 形状语言
- **有机曲线**：记忆碎片、生物、植物 — 暗黑童话的柔软
- **几何碎裂**：建筑、遗迹、现实裂缝 — 宇宙恐怖的冰冷
- **分形递归**：深渊层级、UI 边框、侵蚀纹理 — 无尽深渊的数学恐惧
- **粒子飘散**：星尘、记忆碎片、侵蚀粒子 — 时间流逝的视觉化

## 字体规范
- 标题：衬线体，带细微的"侵蚀"效果
- UI：无衬线体，简洁，高可读性
- 数字/代码：等宽字体，用于深渊层级指示

## UI 设计原则
1. **极简 HUD** — 仅在需要时显示，默认隐藏
2. **环境即 UI** — 用光影和粒子代替传统 UI 元素
3. **侵蚀系统** — 随深度增加，UI 逐渐"被侵蚀"：边框碎裂、文字模糊、颜色偏移
4. **记忆之锚计数** — 用荧光青光点数量表示，而非数字

## Shader 风格参考
- 星空粒子：缓慢飘散的星尘，有深度视差
- 侵蚀效果：噪声驱动的边缘溶解，带荧光青/虚空橙边缘光
- 深渊雾气：多层 Perlin 噪声，随深度变浓变暗
- 记忆回放：VHS 故障效果 + 色彩偏移
- 现实侵蚀：扫描线 + 屏幕震动 + 色温偏移 + 现实裂缝 + 故障爆发，深度驱动渐进异变
- 水面反射：双界面视觉（水上/水下），多层正弦波纹 + 噪声扭曲反射 + 深渊荧光渗透 + 侵蚀叠加
- 深渊脉动：奇点引力场 + 多频呼吸系统（0.1Hz/0.3Hz/1Hz 三频共振）+ 侵蚀脉冲冲击波 + 记忆残响几何碎片
- 深渊记忆回响：记忆波纹扩散 + 碎片浮现（六边形/三角形/菱形）+ 时间扭曲场 + 回声脉冲冲击波
- 深渊星云：多层 domain-warped Perlin 噪声 + 递归扭曲 + 深度分层色彩（蓝→紫→红→黑）+ 侵蚀叠加 + 视差滚动
- 深渊暴风：记忆雨（风场偏转+深度暖色渐变）+ 侵蚀尘埃（涡旋捕获+脉冲呼吸）+ 虚空闪电（分支+全屏闪光+深度≥2触发）+ 涡旋风场（深度驱动+漂移+正弦扰动）
- 深渊裂隙：应力断裂扩展（噪声场路径+多级分支）+ 虚空之眼（裂隙交汇凝聚+多层径向渐变+脉动瞳孔+移动虹膜）+ 维度碎片（六边形/三角形/菱形+噪声场漂移）+ 侵蚀蔓延（距离场+虚空橙渐变+深度驱动）

## 危险区域视觉规范
- 尖刺：三角锥阵列，深度<2荧光青/深度≥2虚空橙，脉冲发光，径向光晕
- 岩浆：黄-橙-红渐变，正弦波表面动画，上升火焰粒子（黄核心→橙→红衰减），径向暖光晕
- 虚空裂缝：活跃期60%（红-暗紫-橙渐变+橙色边框发光+脉冲+扩散波环），休眠期40%（红色虚线警告框），周期2-4秒

## 天气系统视觉规范
- 浅层：微弱记忆雨（荧光青），少量尘埃，无闪电
- 中层：中等记忆雨+尘埃，1个涡旋，无闪电
- 深层：强记忆雨（偏暖色）+密集尘埃+2个涡旋+偶发闪电
- 最深层：暴风记忆雨（虚空橙偏色）+极密尘埃+3个涡旋+频繁闪电

## 小地图 HUD 规范
- 位置：右下角，120x70px，距边缘15px
- 背景：深渊蓝半透明(0.6 alpha)+荧光青细边框(0.15 alpha)
- 平台：星尘灰半透明(0.3 alpha)，移动平台蓝色(0.3 alpha)
- 锚点：荧光青脉冲光点(2px)，危险：岩浆橙/裂缝红/尖刺橙
- 玩家：荧光青核心(2px)+脉冲光晕(5px)+方向指示线(6px)
- 视口：月光白半透明框(0.2 alpha)

## 侵蚀系统视觉规范
- 深度0（浅层）：无侵蚀，纯净童话感
- 深度1（中层）：微弱扫描线 + 轻微暗角 + 玩家青色残影 + 底部微暖色
- 深度2（深层）：明显扫描线(含闪烁条) + 屏幕微震 + 现实裂缝 + 故障爆发(8-15s) + HUD文字乱码 + 王冠闪烁 + 橙红粒子
- 深度3（最深层）：密集扫描线+RGB色差 + 强屏幕震动 + 多条裂缝 + 频繁故障爆发 + 强色温偏移 + 锚点脉冲不稳定 + 眼睛变虚空橙 + 粒子运动不稳定

## 结局场景视觉规范
- 阶段「坠落」：纯黑背景 + 缓慢坠落粒子 + 淡灰文字
- 阶段「揭示」：宇宙之眼（虚空橙椭圆+脉动+多层光晕），现实裂缝从边缘生长
- 阶段「抉择」：双选项框，选中项荧光青边框+发光，未选中星尘灰
- 结局A「回归虚无」：粒子逐渐消失，画面渐暗至纯黑，月光白标题
- 结局B「凝视深渊」：粒子向中心坍缩，之眼扩大，虚空橙标题，裂缝加剧
