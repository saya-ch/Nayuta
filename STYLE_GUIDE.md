## 光照风格
- **核心光源**：生物荧光 — 来自记忆碎片和环境中的微弱发光体
- **环境光**：极低，整体偏暗，营造深渊压迫感
- **特殊光效**：
  - 记忆之锚：脉冲式荧光青发光，有呼吸感
  - 侵蚀效果：边缘泛虚空橙，像现实被烧穿
  - 宇宙存在：不规则的光线扭曲，引力透镜效果
- **光照参考**：深海生物荧光 + 星云摄影 + Hollow Knight 的"光点"系统

## 形状语言
- **有机曲线**：记忆碎片、生物、植物 — 暗黑童话的柔软
- **几何碎裂**：建筑、遗迹、现实裂缝 — 宇宙恐怖的冰冷
- **分形递归**：深渊层级、UI 边框、侵蚀纹理 — 无尽深渊的数学恐惧
- **粒子飘散**：星尘、记忆碎片、侵蚀粒子 — 时间流逝的视觉化

## 字体规范
- 标题：衬线体，带细微的"侵蚀"效果
- UI：无衬线体，简洁，高可读性
- 数字/代码：等宽字体，用于深渊层级指示

## UI 设计原则
1. **极简 HUD** — 仅在需要时显示，默认隐藏
2. **环境即 UI** — 用光影和粒子代替传统 UI 元素
3. **侵蚀系统** — 随深度增加，UI 逐渐"被侵蚀"：边框碎裂、文字模糊、颜色偏移
4. **记忆之锚计数** — 用荧光青光点数量表示，而非数字

## Shader 风格参考
- 星空粒子：缓慢飘散的星尘，有深度视差
- 侵蚀效果：噪声驱动的边缘溶解，带荧光青/虚空橙边缘光
- 深渊雾气：多层 Perlin 噪声，随深度变浓变暗
- 记忆回放：VHS 故障效果 + 色彩偏移
- 现实侵蚀：扫描线 + 屏幕震动 + 色温偏移 + 现实裂缝 + 故障爆发，深度驱动渐进异变

## 侵蚀系统视觉规范
- 深度0（浅层）：无侵蚀，纯净童话感
- 深度1（中层）：微弱扫描线 + 轻微暗角 + 玩家青色残影 + 底部微暖色
- 深度2（深层）：明显扫描线(含闪烁条) + 屏幕微震 + 现实裂缝 + 故障爆发(8-15s) + HUD文字乱码 + 王冠闪烁 + 橙红粒子
- 深度3（最深层）：密集扫描线+RGB色差 + 强屏幕震动 + 多条裂缝 + 频繁故障爆发 + 强色温偏移 + 锚点脉冲不稳定 + 眼睛变虚空橙 + 粒子运动不稳定

## 结局场景视觉规范
- 阶段「坠落」：纯黑背景 + 缓慢坠落粒子 + 淡灰文字
- 阶段「揭示」：宇宙之眼（虚空橙椭圆+脉动+多层光晕），现实裂缝从边缘生长
- 阶段「抉择」：双选项框，选中项荧光青边框+发光，未选中星尘灰
- 结局A「回归虚无」：粒子逐渐消失，画面渐暗至纯黑，月光白标题
- 结局B「凝视深渊」：粒子向中心坍缩，之眼扩大，虚空橙标题，裂缝加剧
- 制作人员：纯黑背景，滚动字幕，标题荧光青发光，角色名月光白，描述星尘灰
